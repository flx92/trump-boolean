# minimal-package

This package is intended to show the bare minimum required to make up a
Node.js package.

## Installation

```
npm install --save minimal-package
```

## Usage

```js
const minimalPackage = require('minimal-package');
console.log(minimalPackage); // 'minimal-package'
```
